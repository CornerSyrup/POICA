# HTML cheatsheet

## 要素

```xml
<element attribute="something">
</element>

<element attribute="something" />
```

```html
<element class="class1" id="element1">
</element>

<element class="class1" id="element2" />
```

## よく使う要素

### 構造

```html
<header></header>
<main></main>
<section></section>
<aside></aside>
<footer></footer>
```

### 一行で終わる

```html
<img src="" alt="" />
<br />
<hr />
```

### 割とよく使う

```html
<div></div>
<table>
    <tr>
        <th></th>
        <td></td>
        <td></td>
    </tr>
</table>
<a href=""></a>
```
