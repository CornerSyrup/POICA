# Style Cheatsheet

## CSS

```css
element {
  key: value;
}

element.class#id {
  key: value;
}
```

## LESS

```less
@margin-large: 10px;
@main-bg: green;

element {
  key: value;
}

element.class#id {
  key: value;
}

.parent-element {
  .chile-element {
    background-color: @margin-large;
    margin: @margin-large;
  }
}
```
